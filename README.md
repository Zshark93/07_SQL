# SQL és MySQL – rövid README

## SQL röviden
Az SQL egy adatbázis-kezelő nyelv.
Arra használjuk, hogy táblákban tárolt adatokat hozzunk létre, lekérdezzünk, módosítsunk és töröljünk.

## Mire használjuk?
- adatok tárolására
- adatok keresésére
- adatok módosítására
- adatok törlésére
- táblák és kapcsolatok létrehozására

## Hol használják?
- weboldalaknál
- nyilvántartó rendszereknél
- iskolai és céges adatbázisoknál
- alkalmazások háttéradat-kezelésénél

## Mit tanulunk benne?
- tábla, mező, rekord
- elsődleges kulcs, idegen kulcs
- SELECT, INSERT, UPDATE, DELETE
- CREATE TABLE és más alaputasítások

## Lényege
Az SQL segít abban, hogy az adatokat rendezetten, gyorsan és hatékonyan tudjuk kezelni.

## Mit érdemes letölteni Windowsra?

### Ajánlott választás
- **MySQL Server:** 8.4.8 LTS – MSI Installer
- **MySQL Workbench:** 8.0.46 – MSI Installer

### Fontos megjegyzés
A MySQL dokumentáció szerint Windows alatt a legegyszerűbb és ajánlott módszer az MSI-alapú telepítés.
A MySQL Installer 8.0.45 még elérhető, de a MySQL 8.1-es verziótól felfelé már a külön termékes MSI telepítést javasolják.
